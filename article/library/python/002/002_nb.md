```python
import sys

a = [i ** 2 for i in range(10000)]

print('(先頭の10個を表示) a :',a[0:10])
print('aの使用メモリ        :', sys.getsizeof(a))
```

    (先頭の10個を表示) a : [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
    aの使用メモリメモリ   : 87624
