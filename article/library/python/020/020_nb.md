このDataFrameから、例えば、itemのカウントが2個以上の項目だけgroupbyしたいという機会があった。
これは以下の様に、groupbyの後にfilterとlambdaを利用する事で実現できる。


```python
df.groupby('item').filter(lambda x: x['sales'].count() >= 2)
```
