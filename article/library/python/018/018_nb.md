## 三つの引数を持つ関数を用意


```python
def test(a,b,c):
  print('a : ', a)
  print('b : ', b)
  print('c : ', c)
```

### リスト型をアンパックして代入する


```python
arg = [
   '123',
   '456',
   '789',
]

test(*arg)
```

    a :  123
    b :  456
    c :  789


### 辞書型をアンパックして代入する


```python
arg = {
  'a': '123',
  'b': '456',
  'c': '789',
}

test(**arg)
```

    a :  123
    b :  456
    c :  789
